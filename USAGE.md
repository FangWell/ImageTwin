# 🚀 快速使用指南

## Windows用户

### 推荐方法（适用于所有Python版本）
1. **双击 `start_simple.bat`** 启动服务
2. **双击 `simple_frontend.html`** 打开Web界面  
3. 使用完毕后**双击 `stop.bat`** 关闭服务

### 如果启动失败
1. **双击 `install_deps.py`** 手动安装依赖
2. 然后使用上述推荐方法启动

### 传统方法（可能在新Python版本中失败）
1. **双击 `start.bat`** 启动服务（首次运行会自动安装依赖）
2. **双击 `simple_frontend.html`** 打开Web界面
3. 使用完毕后**双击 `stop.bat`** 关闭服务

## macOS/Linux用户
1. 打开终端，执行：
   ```bash
   chmod +x start.sh stop.sh
   ./start.sh
   ```
2. **双击 `simple_frontend.html`** 打开Web界面
3. 使用完毕后执行：`./stop.sh`

## ✨ 主要功能

### 1. 索引图片目录
- 在"搜索目录"输入框中输入图片文件夹路径
- 点击"索引目录"按钮（会显示进度条）
- 目录路径会自动保存，下次打开无需重新输入

### 2. 搜索相似图片
- 点击上传区域选择要查找的图片
- 选择后会显示图片预览
- 调整相似度阈值（0.1-1.0，数值越高越严格）
- 勾选高级选项：
  - ✅ 忽略分辨率差异：匹配不同尺寸的相同图片
  - ✅ 忽略图片元数据：忽略拍摄信息等数据
- 点击"🔍 搜索相似图片"开始搜索

### 3. 查看搜索结果
- 每个结果显示相似度百分比
- 点击预览图可查看大图
- 点击"📋 复制路径"复制文件完整路径

## 🛠️ 系统要求
- **Python 3.11+**（启动脚本会自动检测并提示安装）
- **现代浏览器**（Chrome、Firefox、Safari、Edge等）
- **操作系统**：Windows 10+、macOS 10.14+、Ubuntu 18.04+

## ❓ 常见问题

**Q: 启动失败怎么办？**
A: 
1. 首先尝试运行 `start_simple.bat`（推荐）
2. 如果仍然失败，运行 `install_deps.py` 手动安装依赖
3. Python 3.14等新版本可能有兼容性问题，推荐使用Python 3.11-3.12
4. 检查是否安装了Python 3.8+，Windows用户可从python.org安装

**Q: 图片预览显示不了？**
A: 确保后端服务已启动，浏览器支持现代Web标准

**Q: 搜索结果为空？**
A: 确认目录已正确索引，尝试降低相似度阈值

**Q: 如何停止服务？**
A: 运行对应的stop脚本，或在启动窗口按Ctrl+C
